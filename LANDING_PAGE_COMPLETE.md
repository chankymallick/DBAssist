### Update Colors:
Edit CSS variables in `:root`:
```css
--primary-color: #2c3e50;
--secondary-color: #3498db;
--accent-color: #27ae60;
```

### Add Features:
Copy a feature card and edit:

After creating release, update href attributes:
